1. Stack

- Frontend: React, Typescript, Talwind CSS
- Backend: Node.js, Typescript, Fastify
- Database: MongoDB, Prisma (as a tool, for data access)

2. Getting started
   ```javascript
   git clone https://github.com/setlors/blackmarketik.git
   ```

- Set up the server:

  ```javascript
  cd server
  npm install
  npm run dev
  ```

- Set up frontend:

  ```javascript
  cd frontend
  npm install
  npm run dev
  ```
